# 10. Licensing & editions

Aefos AI ships in three editions. The **Community edition is free** — including for
**internal commercial/business** use, with no penalty.

## Editions

| Edition | Price | Includes |
|---------|-------|----------|
| **Community** | Free | The **Chat**, including **Agent mode**, with your own CLI account |
| **Pro** | Subscription | Productivity features and execution power (below) |
| **Enterprise** | Contract | Broad corporate use, support, governance |

### What each tier unlocks

The philosophy is: **the convenience/auto-creation is paid; the manual core is free.**
A Community user can still do everything **by hand** (write the `.mcp.json`, type
commands manually); Pro buys the in-IDE wizards and execution power.

| Feature | Tier |
|---------|------|
| Chat (panel + conversation) | **Community** |
| Agent mode (the chat default) | **Community** |
| Terminal | **Pro** |
| MCP auto-setup/provisioning (dialog) | **Pro** |
| Command-creation wizard | **Pro** |
| Advanced project context | **Pro** |
| Session history + templates | **Pro** |
| Silent reload (smooth AI flow) | **Pro** |

> During **beta**, enforcement is **soft** (the feature works and there is an upsell
> notice). Enforcement becomes strict at general availability (GA).

## How licensing works

- **Node-locked single-seat**: one license covers **one copy** of Delphi per
  machine/user. The server guarantees atomically that **one key = one active IDE**
  (for 1 seat).
- **Self-transfer**: **deactivating** on one machine frees the seat so you can
  activate on another Delphi.
- **Offline-safe**: once validated, the license works **offline** within a grace
  window; without a key, there is a built-in **trial**.

## How to activate

1. In **View → Aefos AI (Chat)**, click the license item (it shows the current
   status, e.g. **License: active**).
2. Enter your **license key**.
3. Done — activation binds this copy of Delphi to your seat.

The license menu item itself shows the current **status** (e.g. *License: active*).

## Free business use

The **Community** edition can be used by companies of any size, internally, **with no
per-seat fee, no user limit, and no penalty**. Details are in the [EULA](../../LICENSE)
(see also [Privacy & license](12-privacy-and-license.md)).
